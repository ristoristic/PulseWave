    打印驱动 	Version 7.222.5412.81

注意事项
======

1. Windows

- 当使用 “Point and Print” 将安装在服务器上的打印驱动安装到客户端 PC 上，然后通过客户端 PC 的打印驱动在[常规]标签的[设置]里设置默认值后，即使对服务器打印驱动的[标准设置]进行更改，默认值也无法反映到客户端 PC 上。

- 在 Windows 8 32 位系统中进行打印时，会弹出非活动状态的对话框，例如：“私密打印密码”和“部门代码”。这是因为 Windows 8 32 位系统中的 WOW64 启用了该应用程序。请在设置前使显示的对话框处于活动状态。

- 当使用“多合一打印”，纵向和横向的图像并存在同一页面上时，将进行缩小打印而不是旋转图像打印。

- 当通过双面打印模式将包含纵向和横向图像的原稿进行打印时，如果图像的方向发生变化，则一个图像会被打印在下一页上。

- 即使点击屏幕右上角的“?” 键，也不会出现帮助信息。

- 在 NetWare 环境下进行打印时，安装打印驱动程序后不能立即进行 SNMP 通信。打开打印驱动程序的“属性”，点击“配置”选项标签和“现在更新”按钮，找到当前使用的 e-STUDIO 系列。

- 当 PS3 打印驱动与 Adobe 字体管理器同时使用时，将导致错误。所以请不要使用 Adobe 字体管理器。当您想使用 Adobe 字体管理器时，请使用 PCL 打印驱动。

- 使用 LPR 打印时，应禁用字节计数。

- 打印水印时，结果可能不像您根据设置值所期望的那样。在这种情况下，需改变水印的大小和位置，然后再次打印水印。水印的位置在可打印区域内才有效。

- 与打印驱动 TOSHIBA Universal Printer、TOSHIBA e-STUDIO Series PS3、TOSHIBA e-STUDIO Series XPS 的兼容性
  用 TOSHIBA Universal Printer、TOSHIBA e-STUDIO Series PS3、TOSHIBA e-STUDIO Series XPS 创建的“配置文件”和“水印”、“叠印”、“导出”的设置文件在该版本的打印驱动上无法使用。
  请重新创建文件。

- 使用配置文件时的注意事项
1. <安装/未安装硬盘(HDD)的机型之间的兼容性>
1.1. 当在安装 HDD 的机型上，使用未安装 HDD 机型所创建的配置文件时，将无法选择除“正常打印”和“打印到叠印文件”之外的打印作业。
请通过取消来关闭打印驱动程序的 GUI 后，再重新打开。

1.2. 当在未安装 HDD 的机型上，使用安装 HDD 机型所创建的配置文件时，包括需要使用 HDD 在内的所有打印作业都可以选择。
但请不要选择需要使用 HDD 的打印作业。如果发送需要使用 HDD 的打印作业，该作业将会在多功能复合机上被删除。
有关需要使用 HDD 的打印作业的详细情况，请参照《打印功能手册》。


2. <安装/未安装自动双面单元(ADU)的机型之间的兼容性>
2.1. 当在安装 ADU 的机型上，使用未安装 ADU 机型所创建的配置文件时，将无法选择双面打印。
请通过取消来关闭打印驱动程序的 GUI 后，再重新打开。

2.2. 当在未安装 ADU 的机型上，使用安装 ADU 机型所创建的配置文件时，可以选择双面打印。
但请不要选择双面打印。如果发送双面打印作业，该作业将会被打印在单面上。

- 从应用程序打印。

 (1) 概述
     虽然有些打印选项可以在应用程序中设定，但是不要使用应用程序来设置打印选项。应该使用打印驱动程序来设置打印选项。
     在应用程序中设置了打印选项后进行打印时，您最终得到的结果可能不是您所预期的，或者系统需要较长的时间来进行处理。
     不要在应用程序中设置“逐份打印”。
     在应用程序中设置纸张输出托盘时，可能会显示两个“自动选择输出托盘”，此时，请选择下方的“自动选择输出托盘”。

 (2) Microsoft Excel
     Excel 把每个工作表作为一项打印作业。打印驱动程序的设置仅应用于第一个工作表。
     当您在 Excel 的打印对话框的［份数］中输入大于 1 的数字时，不要选择［逐份打印］。

 (3) Adobe Acrobat
     当使用 PS3 打印时，打印的图像可能被放大或者缩小。在这种情况下，使用 PCL6 或者使用具有如下设置的 PS3，可以按适当尺寸打印图像。
     打开“打印”对话框为 PS3 执行以下设置。
     - 从［页面缩放方式］菜单中选择［适合纸张］。
     - 选中［按 PDF 页面大小选择纸张来源］
    （注意：当使用 PDF 所设置的纸张尺寸时，打印驱动的纸张设置将无效。）

 (4) Adobe PageMaker
     对于私密打印和部门代码打印，请使用 Adobe PageMaker 的 WindowsPPD，一定要指定一个代码，否则将会打印一个错误打印页。


- Windows 服务器的 Point & Print

在服务器的“打印机默认值”中设置的打印设置不能反映在客户计算机上，该情况视服务器的操作系统和客户计算机的操作系统的组合条件而变。


- 蓝牙
  该多功能复合机可能不能使用一些商业上用的蓝牙 USB 适配器。
  有关蓝牙 USB 适配器，请参考蓝牙的操作手册。

打印
--------------------------------------------------------------------------------

注意事项
======

1. 有些文件可能会因为分配给打印的内存不足而无法打印。

2. 在彩色模式下打印时，灰度区域由黑色墨粉打印。这是由于打印驱动程序中，“图像质量”标签的“纯灰”选项默认设置为启用。因此，根据打印数据的不同，可能导致不正确的输出效果（如平滑等级会出现波纹等）。在这种情况下，不要选中“纯灰”选项。

3. 当使用 TOSHIBA Universal Printer 2 打印驱动从设备上（计算机，平板电脑等）发送打印作业，并指定“通用机型”为打印机名称时，输出的作业可能与打印驱动中指定的打印设置/选项不同。（例如彩色/黑白、纸张来源、托盘选择、纸张尺寸/纸质类型、装订和打孔等打印设置/选项可能在使用默认设置打印作业时与指定的不同。）
